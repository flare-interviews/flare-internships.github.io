The objective of your assignment is to write an application that will sort through chat messages and display them to the user. We only want to display messages from a specific time range, and we want to print them in order. Chat messages can be fetched from a web service that returns a JSON payload. The dates used to filter the messages have to be parsed from an HTML page displayed on a website.

  • To get the list of messages, a request can be made to the following endpoint: “https://flare-interviews.github.io/flare-internships.github.io/data/messages.json”. It will return a payload with the following format:

[

  {

    "user": "Mark",

    "message": "Hello, is anyone here?",

    "timestamp": 1642037302

  },

  {

    "user": "Tommy",

    "message": "Oh, hi Mark!",

    "timestamp": 1683045971

  },

  {

    "user": "Martin",

    "message": "I'm from the future!",

    "timestamp": 2337073245

  }

]
  • To get the dates used to filter the messages, a request can be made to the following endpoint: “https://flare-interviews.github.io/flare-internships.github.io/data/parameters.html”. It returns some HTML where the parameters can be found.
    • Bonus: this web page doesn’t like being queried by crawlers. It will only return a successful response if it thinks the request is coming from a browser

<html>

  <body>

    <div id="parameters">

      <p>Start: August 10, 2013</p>

      <p>End: January 11, 2024</p>

    </div>

  </body>

</html>
  • We want an output formatted in a fashion similar to a chat log, with messages filtered according to the start and end time, in a chronological order. For example:

[Jan 10th, 2023 11:23 AM] Mark: Hey everyone!

[Jan 10th, 2023 11:26 AM] Tommy: Oh, hi Mark!
